# 🆘 When something breaks (it will — that's normal)

Nothing here is scary. Almost everything is fixed the same way: **tell Claude
what happened and let it fix it.** You built this by talking; you fix it by
talking too.

## The one move that fixes most things

If you see a scary red message anywhere, copy the whole thing and paste:

> 📋 **Paste this to Claude (with the message):**
```text
Something went wrong and I got this — please explain it simply and fix it for me:

[paste the entire message here]
```

Then try again. Repeat if needed. This back-and-forth **is** building. Stay calm.

---

## Common hiccups

**"My browser link won't open / the page won't load."**
Your agent probably isn't running. Paste to Claude: *"start my agent in the
background and give me the link."* Then click the fresh link.

**"It says the address is already in use."**
Something's already running on that door. Paste: *"my agent's port is busy —
start it on a different port and give me the new link."*

**"It doesn't sound like me."**
That's not a bug — it's the personality. Paste: *"update my PERSONALITY.md to be
more [funny / casual / blunt / warm]"* and test again. (No restart needed.)

**"command not found" / "not recognized."**
A tool isn't installed yet, or the terminal needs a restart. **Close your editor
and open it again**, then try once more. Still stuck? Grab a mentor.

**"It stopped replying / usage limit reached."**
Switch to a lighter model — type `/model sonnet` to Claude — and keep test
messages short. Limits refill over time, so take a short break if needed.

**"It can't read my email / calendar."**
That connector isn't switched on. Go to **claude.ai/customize/connectors**,
connect **Gmail** (or **Google Calendar**), then try again in a fresh chat.

**"My reminder never popped up."**
Your agent has to be running for reminders to fire. Make sure it's still going
(*"is my agent still running?"*) and that you allowed notifications on your laptop.

**"I think I broke everything."**
Don't panic. Paste: *"take me back to my last checkpoint — the last working
version."* That's what the checkpoints were for.
